================================================================================
主要模块说明
================================================================================

【RTL核心模块】
- cpu_core_top.v        CPU核心顶层
- if_stage.v            取指阶段
- id_stage.v            译码阶段
- rn_stage.v            重命名阶段
- is_stage.v            发射阶段
- ex_stage.v            执行阶段
- mem_stage.v           访存阶段
- wb_stage.v            写回阶段
- rob.v                 重排序缓冲区（32项）
- rat.v                 寄存器别名表
- reservation_station.v 保留站
- prf.v                 物理寄存器堆（64个）
- alu_unit.v            ALU单元
- mul_unit.v            乘法器（3周期流水线）
- div_unit.v            除法器（最多34周期）
- branch_unit.v         分支单元

【分支预测单元】
- bpu.v                 BPU顶层
- tage_predictor.v      TAGE预测器
- bimodal_predictor.v   双模态预测器
- btb.v                 分支目标缓冲（256项）
- ras.v                 返回地址栈（8项）
- loop_predictor.v      循环预测器

【存储子系统】
- icache.v              指令缓存（4KB，4路）
- dcache.v              数据缓存（4KB，4路，写回）
- lsq.v                 Load/Store队列
- load_queue.v          加载队列（8项）
- store_queue.v         存储队列（8项）
