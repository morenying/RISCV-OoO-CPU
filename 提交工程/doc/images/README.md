# RISC-V 乱序执行处理器 · 项目介绍图集

本目录包含 RISC-V RV32IM 乱序执行处理器项目的介绍图片，采用绿蓝极光银河星空风格设计，楷书字体。

## 图片列表

| 序号 | 文件名 | 内容描述 |
|------|--------|----------|
| 01 | `01_项目总览.svg` | 项目整体介绍，包含七级流水线概览、核心组件和性能目标 |
| 02 | `02_流水线架构.svg` | 七级流水线详细架构，每个阶段的功能和数据流说明 |
| 03 | `03_分支预测单元.svg` | BPU 分支预测单元架构，包含 TAGE、BTB、RAS 等组件 |
| 04 | `04_乱序执行引擎.svg` | 乱序执行核心组件：ROB、RAT、保留站、PRF、CDB |
| 05 | `05_存储子系统.svg` | 缓存系统和访存队列：I-Cache、D-Cache、LSQ、AXI 总线 |
| 06 | `06_指令集支持.svg` | RV32IM + Zicsr 指令集详细说明和指令格式 |

## 设计风格

- **主题**: 绿蓝极光银河星空
- **字体**: 楷书 (KaiTi, STKaiti)
- **配色**: 
  - 主色调: 青绿色 (#00ffcc)、天蓝色 (#00aaff)
  - 辅助色: 金黄色 (#ffcc00)、紫色 (#aa88ff)、粉色 (#ff88ff)
- **特效**: 
  - 星空背景
  - 极光光带
  - 发光效果 (glow filter)
  - 半透明背景框

## 使用方式

这些 SVG 图片可以：
- 直接在浏览器中打开查看
- 嵌入到 Markdown 文档中
- 用于项目演示和文档
- 导出为 PNG/PDF 等格式

## 技术规格

- 格式: SVG (可缩放矢量图形)
- 尺寸: 1200 x 800~900 像素
- 编码: UTF-8
- 兼容性: 现代浏览器
