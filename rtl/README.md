# RTL Source Files

This directory contains all Verilog RTL source files for the RISC-V Out-of-Order CPU.

## Directory Structure

- `core/` - CPU core modules (pipeline stages, functional units)
- `cache/` - Cache subsystem (I-Cache, D-Cache)
- `bpu/` - Branch Prediction Unit (TAGE, BTB, RAS)
- `mem/` - Memory subsystem (LSQ, memory interfaces)
- `common/` - Common modules and utilities
