# Implementation Plan: RISC-V Out-of-Order CPU

## Overview

本实现计划将RISC-V乱序执行CPU的设计分解为可执行的编码任务。采用自底向上的方式，先实现基础模块，再逐步集成到完整系统。使用Verilog 2001标准编写可综合代码。
